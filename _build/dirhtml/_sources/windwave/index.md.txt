## Wind and wave data

This is a markdown page instead of a jupyter notebook

How to export a plotly chart:
`fig.write_html("piechart.html")`

To render the html, you can save it on the same folder of your index.html of your page and then include the following code on the index.html:

`<iframe src="piechart.html" width="100%" height="500" style="border:1px solid black;">  </iframe>`
